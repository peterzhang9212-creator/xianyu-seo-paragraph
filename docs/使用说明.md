# 使用说明

## 1. 环境准备

- 安装 Python 3.12 或更高版本
- 安装依赖：

```powershell
python -m pip install -r requirements.txt
```

- 如果要跑发布步骤，还需要：
  - 本机可正常打开 `BitBrowser`
  - 已登录闲鱼账号
  - 页面可正常访问

## 2. 启动程序

在项目目录运行：

```powershell
python app.py
```

## 3. 四步流程

### 文件组装

- 选择城市表 Excel
- 选择输出父目录
- 填写固定后缀
- 选择封面图目录
- 选择详情图目录
- 点击 `开始执行`

结果：

- 自动创建商品素材文件夹
- 自动生成 `商品主图.jpg`
- 自动生成 `商品标题.txt`
- 自动复制详情图

### 加水印

- 选择素材目录
- 填写水印文字
- 可调整字号倍率、间距系数、透明度
- 点击 `开始执行`

固定规则：

- 主图和详情图都会加水印
- 原图不会被覆盖
- 新图输出到独立目录，命名规则是 `原目录名_水印`

### 生成上传表

- 选择正文模板文件
- 选择素材目录
- 选择上传表输出位置
- 点击 `开始执行`

规则：

- 正文里的 `导出商品名` 必须和素材子文件夹名一致
- 不一致时会直接报错，不会自动改名

### 发布设置

- 选择上传 Excel
- 确认规格名、价格、库存、起始网址
- 可先点 `执行预检`
- 确认本机 `BitBrowser` 已打开后，再点 `开始发布`

### 子文件夹工具

- 选择一个根目录
- 选择导出 Excel 位置
- 点击 `开始导出`

用途：

- 单独导出一级子文件夹名称
- 方便你人工判断某一批目录是否要继续参与后续流程

## 4. 自动带入规则

- 程序会记录上一次输入的路径和参数
- 文件组装完成后，会记录最近一次组装输出目录
- 加水印默认优先读取最近一次组装输出目录
- 生成上传表默认优先读取最近一次水印输出目录；没有时回退到最近一次组装输出目录
- 发布设置默认优先读取最近一次生成的上传 Excel
- 子文件夹工具会记录最近一次导出的根目录和 Excel 路径

## 5. 可以跳步使用

- 不强制按 1 到 4 顺序执行
- 可以直接选一个现成素材目录去加水印
- 可以直接选一个现成素材目录去生成上传表
- 可以直接选一个已有上传 Excel 去做发布预检或发布

## 6. 内置模板

正文模板文件位置：

`templates/content_template.xlsx`

建议每次复制这份模板，再填写本次内容。

## 7. 常见问题

### 正文商品名和文件夹名不一致

处理方式：

- 检查正文模板里的 `导出商品名`
- 检查素材目录下的子文件夹名
- 两边必须完全一致

### 主图匹配不到

处理方式：

- 确认封面图文件名和城市表中的序号一致
- 例如城市表里是 `001`，封面图也应是 `001.jpg`

### 发布预检提示缺主图

处理方式：

- 确认每个商品文件夹内都有 `商品主图.jpg`

### 发布失败

优先检查：

- `BitBrowser` 是否已打开
- 是否已登录闲鱼
- 页面结构是否有变化
- 上传 Excel 中素材目录是否存在
